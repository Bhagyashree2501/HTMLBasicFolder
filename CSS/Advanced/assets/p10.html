CSS PSEUDO-CLASSES..!!

The CSS Pseudo-classes allow you to style the dynamic states os anelement such as hover, active andnfocus state, as well as elements that are existing in the document tree but can't be targeted via the use of other selectors without adding any ID's or classes to them, for example, targetting the first or last child elements.

A pseudo-class starts with a colon (:). Its syntax can be given with:

Selector:pseudo-class {property:value;}
