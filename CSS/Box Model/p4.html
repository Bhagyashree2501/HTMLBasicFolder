CSS Box Model..!!

Every element that can be displayed on a web page is comprised of oneor more rectangular boxes.css box model typically describes how theses rectangular boxes are laid out on a web page. These boxes can have different properties and can interact with each other in different ways,but every box has a content area and optional surrounding padding, border and margin areas.
